# Projet : Évaluateur et Typeur de Lambda-Calcul

## Description
Ce projet consiste en un évaluateur et un typeur pour un $λ$-calcul étendu, permettant de réduire des expressions lambda, effectuer des substitutions, des conversions alphabétiques, et vérifier la validité des types. Le projet gère plusieurs constructions, notamment les combinateurs, les listes, les termes conditionnels et les opérateurs arithmétiques.

## Prérequis
Pour exécuter ce projet, vous devez disposer d'OCaml installé sur votre machine.

## Instructions pour lancer le projet
1. Utilisez la commande `make` pour compiler le projet :
   ```sh
   make
   ```

2. Chargez les fichiers nécessaires dans l'interpréteur OCaml en utilisant la commande :
   ```sh
   make run
   ```

Cela affichera les résultats des différents tests d'évaluation, de substitution, et de typage des termes lambda définis dans le projet.
